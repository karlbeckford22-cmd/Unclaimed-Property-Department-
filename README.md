# Unclaimed Property Department Website

This project is the official website for the **Unclaimed Property Department**, designed to help users locate and claim unclaimed or abandoned financial assets. The site provides a user-friendly platform for searching, learning, and submitting claims for property such as uncashed checks, bank accounts, and insurance proceeds.

## Features

- **Owner Search:** Easily search for unclaimed property by name or location.
- **Claim Submission:** Submit claims securely online to recover your property.
- **Information & Resources:** Access guidelines, FAQs, and legal requirements related to unclaimed property.
- **Public Awareness:** Educates users about unclaimed property rights and reporting responsibilities.
- **Responsive Design:** Works on desktop, tablet, and mobile devices.
